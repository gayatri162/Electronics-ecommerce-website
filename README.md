# 🛒 Tech Stack - Electronics E-Commerce Website

A responsive electronics e-commerce website developed using **HTML, CSS, and JavaScript** as part of my internship project. The application provides a modern shopping experience with product browsing, filtering, searching, and cart management.

## 🚀 Features

- Responsive design for desktop and mobile
- Product listing with attractive UI
- Category-based filtering
- Product search functionality
- Product details page
- Add to Cart
- Increase/Decrease product quantity
- Remove items from cart
- Dynamic cart total calculation
- Toast notifications
- Smooth animations and hover effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
